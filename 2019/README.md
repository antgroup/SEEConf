# 第二届 SEE Conf 2019 精彩回顾

2019年1月5日，第二届蚂蚁金服体验科技大会 SEE Conf 2019 于杭州蚂蚁 Z 空间成功举办，数位讲师、嘉宾为大家带来了一场体验科技的分享盛宴，近 500 位参会者到场参与此次体验科技盛宴，近 2 万人在线收看直播。无论是 Ant Design 的情感化设计，还是数据可视化之 Story Telling，以及 workshop 中的 Ant Design 资产一起造和 Ant Designers 圆桌沙龙，相信都让大家印象深刻。

大会的相关资料已整理完毕，欢迎大家前往查看：
